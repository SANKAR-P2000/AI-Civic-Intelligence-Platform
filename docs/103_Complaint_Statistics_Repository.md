# Complaint Statistics Repository

## Project

AI Civic Intelligence Platform (AICIP)

---

# Objective

Create repository methods for dashboard statistics.

---

# Methods

```java
long count();

long countByStatus(ComplaintStatus status);

long countByCategory(ComplaintCategory category);
```

---

# Purpose

These methods allow Spring Data JPA to generate SQL queries that calculate complaint statistics directly in the database.

---

# Generated Queries

## Total Complaints

```sql
SELECT COUNT(*)
FROM complaints;
```

## Count by Status

```sql
SELECT COUNT(*)
FROM complaints
WHERE status = ?;
```

## Count by Category

```sql
SELECT COUNT(*)
FROM complaints
WHERE category = ?;
```

---

# Benefits

- No manual SQL
- High performance
- Automatically generated by Spring Data JPA
- Ideal for dashboard analytics
